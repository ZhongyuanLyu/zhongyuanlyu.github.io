Jacob Mattingley (jacobm@stanford.edu)

I am a posdoctoral research scientist in the Data Science Institute at Columbia University. My research interests center around matrix and tensor data analysis in high-dimensional setting, specifically revolving around the fields of mixture models, latent variable models and network models.

Download jemdoc.

Version 0.7.3 was released on 2012-11-27 with some small bug fixes. See the release notes.

Contribute to jemdoc on github.

Goals

  • Simple, consistent syntax.

  • \(\mbox{\LaTeX}\) equation support.

  • Table support.

  • Portability. The (single) jemdoc Python file + single css file + your input file → html.

  • Based on CSS so formatting specifics are independent of jemdoc.

  • Production of clean, standards-compliant XHTML 1.1.

  • Minimal bells and whistles, but simple fallback to raw html if required.
